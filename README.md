# Forensic Officer, Cyber Fair Election Support Team, Gwangju Metropolitan City Election Commission 2026

> Repository for learning, practice, and documentation of forensic activities in the Cyber Fair Election Support Team, Gwangju Metropolitan City Election Commission, 2026.

---

## 📌 목적 (Purpose)
이 저장소는 2026년 광주광역시 선거관리위원회 사이버공정선거지원단 포렌식 요원으로 활동하며  
배운 지식, 실습 내용, 사례 분석 등을 정리하기 위해 만들어졌습니다.  
개인 학습용, 기록용, 향후 참고 자료용으로 활용합니다.

## 🧠 학습 내용 (Learning Topics)
- 디지털 포렌식 기본 원리
- 네트워크 및 로그 분석
- 데이터 복구 및 증거 확보 방법
- 전자 선거 시스템 이해
- 보안 툴 사용법 (예: FML, DFT, DAS, EnCase)

## 💻 실습 기록 (Practical Exercises)

이 섹션에서는 2026년 광주광역시 사이버공정선거지원단 포렌식 요원으로서  
실습할 도구와 활동을 기록합니다. 주로 **모바일 포렌식과 CCTV 포렌식**을 중심으로 실습합니다.

### 1. 모바일 포렌식
- **도구 활용**: FML, DFT, DAS, EnCase 등
- 스마트폰 데이터 추출: 메시지, 통화 기록, 앱 데이터, 위치 정보
- 삭제 데이터 복구 및 해시값 검증
- 모바일 디바이스 이미지 생성 및 분석
- 분석 보고서 작성 및 증거 정리

### 2. CCTV 포렌식
- **도구 활용**: EnCase, DAS 등 영상/저장 매체 분석 가능 도구
- CCTV 영상 데이터 확보 및 원본 이미지 보존
- 영상 프레임 분석 및 이벤트 기록 추출
- 저장 매체(녹화기, SD 카드 등) 데이터 복구 및 검증
- 증거 기반 사건 기록 작성

> 각 실습 내용은 `/practice/mobile`과 `/practice/cctv` 폴더에 정리하며,  
> 스크린샷, 분석 노트, 보고서 등을 함께 기록할 예정입니다.
